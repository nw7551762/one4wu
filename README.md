# one4wu

one4wu 是一個基於 Spring Boot 框架開發的外包購物網站。

快速開始
要在本地運行專案，您需要遵循以下步驟：

1. 複製存儲庫到本地。
2. 打開 src/main/resources/application.properties 檔案。
   尋找並修改相關的設定，例如資料庫連接帳號和密碼。
2. 在命令行中導航到專案目錄。
3. 運行 mvn clean package 命令編譯專案。
4. 運行 java -jar target/universal-house.jar 命令運行專案。

資料庫設定

1. 創建一個新的 MS SQL 資料庫
2. 開啟 MS SQL Server Management Studio
3. 使用 SQL Server 驗證進行身份驗證並登入
4. 選擇 “New Query” 並打開 createDB.sql 檔案 (檔案於根目錄)
5. 運行 createDB.sql 中的 SQL 命令，這會創建您指定的資料庫
